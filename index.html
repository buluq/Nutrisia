Nutrisia: WordPress plugin for Java Multi Mandiri websites

Nutrisia is a WordPress plugin to enhance the functionality of Java Multi Mandiri websites, specificaly WordPress based websites.

Nutrisia is written and crafted for Java Multi Mandiri websites in mind, so I do warn you that it may be will not able to run smoothly on your site. But, I try to write this plugin so it can be installed in any general WordPress-based sites.
